Le <a href="/">Lorraine JUG</a> (Java User Group de Lorraine) organise son premier Coding Dojo!

Quelques prérequis pour la soirée :
<ul>
	<li>Votre machine</li>
	<li>Votre ide préféré</li>
	<li>Maven/Gradle</li>
	<li>Idéalement, Junit et AssertJ dans votre repo local</li>
</ul>
