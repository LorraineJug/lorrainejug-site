    Découvrez Ionic Framework, un SDK front-end gratuit et open source pour développer facilement des applications mobiles hybrides avec HTML5. Ionic utilise les technologies du web que vous aimez pour vous offrir une librairie, construite avec Sass et AngularJS, de composants HTML, CSS et JS optimisés pour le mobile ainsi que tous les outils pour construire des apps interactives.

    Basé sur le principe du "write once and run anywhere", vous codez une seule fois et votre application se lance sur Android, iOS et même Windows Phone. Ionic tire partie de l'appareil sur lequel est lancée votre application grâce à Cordova qui l'encapsule et qui fournit des plugins permettant l'accès natif au matériel et aux api spécifiques de la plateforme.

    Ionic est un écosystème mettant à votre disposition une suite d'outils et de ressources de développement ce qui en fait un package pour le développement mobile complet et le meilleur moyen pour construire des application mobiles hybrides.
